# FilmyKhazana

A clean, modern interface to discover and watch movies and TV shows.

Open at **https://imaryansharma.github.io/FilmyKhazana/**

## What it does

- Browse trending titles, popular TV shows, top rated, new releases, and a curated movies row
- Search the entire TMDB catalog with debounced, URL-synced results
- Save titles to a personal watchlist and pick up where you left off via the Continue Watching shelf
- Open any title for full details, cast, and "more like this" suggestions
- Play in the browser via a multi-server picker — Aurora, Echo, Nova, Polaris and others
- Switch between dark and light mode, on every device
- Works as nicely on a phone as on a 4K display
